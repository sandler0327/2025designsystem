# Typography 字體
>定義介面中的字型、段落以及間距，確保資訊的可閱讀性。
## 網頁版標題、段落
字體：思源黑體、微軟正黑體、Arial<br>
單位：16px=1rem<br>
字重(強調)：500(700)<br>
行距：1.5rem<br>
字距：normal

單位：16px=1rem<br>
字重(強調)：500(700)<br>
行距：1.5rem<br>
字距：normal
